    Get your medical practice management system up and running in minutes.
  </p>

  <Steps>
    <Step title="Install dependencies">
      Clone the repository and install the required packages using npm.
      
      ```bash
      git clone https://github.com/RCasillasV/sb1-16uucj1h.git
      cd sb1-16uucj1h
      npm install
      ```
    </Step>
    
    <Step title="Configure environment variables">
      Set up your Supabase connection and other environment variables in a `.env` file.
      
      ```bash
      VITE_SUPABASE_URL=your_supabase_url
      VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
      VITE_MAX_FILE_SIZE_MB=10
      VITE_BUCKET_NAME=your_bucket_name
      ```
      
      <Note>
        See the [Configuration](/configuration) guide for detailed environment variable setup.
      </Note>
    </Step>
    
    <Step title="Run the development server">
      Start the Vite development server to preview your application locally.
      
      ```bash
      npm run dev
      ```
      
      Your application will be available at `http://localhost:5173`
    </Step>
    
    <Step title="Create your first user">
      Navigate to the registration page and create an administrator account to access all features.
      
      <Tip>
        The first user created will have full administrative privileges to set up your practice.
      </Tip>
    </Step>
  </Steps>
